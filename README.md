INJECTTEST : SIMPLE TEST INJECT PARAMETER FOR WEB VULNERABILITY

Injecttest is simple tool for inject web parameter for test vulnerability (GET/POST)

need:<br>
```
sudo gem install lolcat
sudo apt install curl
```

usage:<br>
```
bash injecttest.sh
```
